# TCP 에코 소보 만들기

### 패킷 구조

| 필드 명     | 타입   | 설명                           | 크기   |
| ----------- | ------ | ------------------------------ | ------ |
| totalLength | int    | 메세지의 전체 길이             | 4 Byte |
| handlerId   | int    | 요청을 처리할 서버 핸들러의 ID | 2 Byte |
| message     | string | 메세지                         | 가변   |
